Drug Interaction Risk Explorer

Two-layer GNN · DrugBank relation type + TwoSides side effects · single shared molecular encoder
📄 IEEE paper ⚙ GitHub 🤗 HF Space ⚠ research use only

L1 DrugBank — what kind of interaction?

86-class softmax over the DrugBank DDI relation types (e.g. metabolism slowdown, increased serum concentration, QTc prolongation). Trained on 191 808 drug pairs · warm-split top-1 = 96.39 %, top-3 = 99.69 %, macro-F1 = 93.20 %.

→ top-K most likely relation types

L2 TwoSides — which polypharmacy side effects might co-occur?

Multi-label sigmoid over the filtered polypharmacy side-effect vocabulary. Built on the frozen Layer 1 encoder so DrugBank top-1 is unaffected. Sweep over frequency thresholds (200 / 500 / 1000 / 2000) — see sweep report for per-threshold AUROC / AP.

Companion to IEEE Xplore document 11428029. Code: github.com/kareemindata/ddi-risk-explorer. Demo: huggingface.co/spaces/kareem-khaled/ddi-risk-explorer.
